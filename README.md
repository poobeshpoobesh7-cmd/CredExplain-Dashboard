
# 💳 CredExplain Dashboard — XAI Loan Approval System

A full-stack explainable AI web application for loan approval 
decisions, featuring a React dashboard with real-time SHAP 
explanations and model insights.

## 🚀 Features
- Real-time loan approval prediction
- SHAP-based explainability visualizations
- Decision Distribution & Applications Over Time charts
- Model Insights panel
- Admin Panel
- Glassmorphism dark UI design

## 🛠️ Tech Stack
**Frontend:** React, Tailwind CSS, Recharts, Vite  
**Backend:** Python, Flask, XGBoost, LightGBM, SHAP  
**Database:** SQLite  

## ⚙️ How to Run

### Backend
```bash
cd backend
pip install -r requirements.txt
python app.py

Frontend
cd frontend
npm install
npm run dev

Open browser → http://localhost:3000
